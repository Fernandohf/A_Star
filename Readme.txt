REQUISITOS

O programa foi desenvolvido em Python 3.6.3 e necessita das seguintes bibliotecas não nativas do Python:
-Numpy
-wxPython 

ARQUIVOS

As funções que implementam o algoritmo A* em si estão no arquivo Aestrela.py

A interface é feita no arquivo UI.py, é a partir dela que as funções do arquivo anterior são chamadas.

O arquivo Aestrela.wxg foi o arquivo de interface criado pelo programa wxGlade. http://wxglade.sourceforge.net/docs/intro.html#program-windows
A partir dele é criado o código compatível com a bilioteca wxPython para a interface.
